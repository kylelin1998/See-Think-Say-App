**见思千言**是基于ChatGPT的跨平台软件。 目前支持Android、Windows和Linux系统，你可以在任何地方运行它。该软件的用户界面简洁，专注于聊天。

关于该软件的设置，你可以选择是否通过代理请求ChatGPT API，此外，你还可以自定义GPT模型的设置。

这个应用程序目前只请求ChatGPT Api和Github来请求更新数据，没有其他请求。
